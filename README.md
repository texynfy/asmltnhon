# Website WordPress - Bán dụng cụ cầu lông 

## Giới thiệu
Đây là dự án website được xây dựng bằng WordPress trong quá trình học tập.

## Chức năng
- Trang chủ
- Trang sản phẩm
- Trang liên hệ
- Trang thanh toán
- Tìm kiếm & lọc
- Bảo mật: HTTPS, chống spam, bảo vệ tài khoản.
- Quản trị (Admin): Quản lý nội dung, người dùng, đơn hàng, thống kê.
- SEO & chia sẻ: Tối ưu Google, chia sẻ MXH, thống kê truy cập.
